Program 6b Write a Python program to illustrate liner plotting with line formatting using Matplotlib.

Description: This program demonstrates a basic implementation of an array traversal.

Code:

            matplotlib.pyplot as plt 
            import pandas as pd 
            import numpy as np 
             
            X=np.linspace(1,20,10) 
            print(X) 
             
            Y2=np.square(X) 
            print(Y2) 
             
            # Line plot 
            plt.plot(X,Y2,marker='X',linestyle='dashed',color='red',linewidth=2,markerfacecolor='blue',markersize=8) 
             
             
            # Add title to the plot 
            plt.title('line plot') 
            plt.xlabel('Numbers') 
            plt.ylabel('Square Root') 
            plt.grid() 
             
            # Display the plot 
            plt.show()
