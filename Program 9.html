Program 9: Write a Python program to draw 3D Plots using Plotly Libraries.

Description: This program demonstrates a basic implementation of an array traversal.

Code:

          Scatter Plot
        import plotly.express as px
        import pandas as pd
        import plotly.express as px
        size = [10, 20, 30, 40, 50]
        x = [1, 2, 3, 4, 5]
        y = [4, 5, 6, 7, 8]
        z = [7, 8, 9, 10, 11]
        fig = px.scatter_3d(x=x, y=y, z=z, size=size)
        fig.show()
