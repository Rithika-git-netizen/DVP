Program 2a:2.a) Defined as a function F as Fn = Fn-1 + Fn-2. Write a Python program which accepts a value for N (where N >0) as input and pass this value to the function. Display suitable error message if the condition for input value is not followed.

Description: This program demonstrates a basic implementation of an array traversal.

Code:

def Fibonacci(n): 
    if n==0 or n==1: 
        return 1 
    else: 
        return Fibonacci(n-1) + Fibonacci(n-2) 
     
num = int (input ("Enter the value of N:")) 
if num>0: 
    fib = Fibonacci(num) 
    print ("Fibonacci value at nth position is:", fib) 
else: 
    print ("Incorrect value of N")
