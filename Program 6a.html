Program 6a Write a Python program to illustrate Linear Plotting using Matplotlib.

Description: This program demonstrates a basic implementation of an array traversal.

Code: import matplotlib.pyplot as plt import pandas as pd import numpy as np X=np.linspace(1,20,10) print(X) Y1=X Y2=np.square(X) print(Y2) Y3=np.sqrt(X) # Line plot plt.plot(X,Y1,'r',X,Y2,'b',X,Y3,'g') plt.legend(['X','X-square','Square-root']) # Add title to the plot plt.title('line plot') plt.grid() # Display the plot plt.show()
