Program 3b Write a Python program to find the string similarity between two given strings

Description: This program demonstrates a basic implementation of an array traversal.

Code: 

            import difflib 
             
            def string_similarity (str1, str2): 
                result=difflib.SequenceMatcher(a=str1.lower(), b=str2.lower()) 
                return result.ratio () 
             
            str1 = input ("Enter the first string :") 
            str2 = input ("Enter the second string :") 
            print ("Original string:") 
            print(str1) 
            print(str2) 
            print ("Similarity between two said strings:") 
            print (string_similarity(str1, str2))
