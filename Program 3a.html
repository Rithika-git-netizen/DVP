Program 3a: Write a Python program that accepts a sentence and find the number of words, digits, uppercas e letters and lowercase letters

Description: This program demonstrates a basic implementation of an array traversal.

Code:

           
str=input ("Enter the Sentence ") 
w, d, l, u=0,0,0,0 
words=str.split () 
w=len (words) 
for i in str: 
    if i.isdigit(): 
        d=d+1 
    elif i.isupper(): 
        u=u+1 
    elif i.islower(): 
        l=l+1 
print ("Number of words", w) 
print ("Number of Digits", d) 
print ("Number of Uppercase Letters", u) 
print ("Number of Lowercase Letters", l)
